## 克隆教程
1. **选择位置**：打开终端，`cd`到你的目标文件夹`目标文件夹`
```
cd 目标文件夹的目录/目标文件夹
```
2. **克隆仓库**：选择你需要的仓库，复制仓库链接`仓库链接`，输入以下指令
```
git clone 仓库链接
```
